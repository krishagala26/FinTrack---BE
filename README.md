# FinTrack Backend

## Overview
FinTrack Backend is the REST API powering the FinTrack Personal Finance Manager application. It handles authentication, transaction management, budget tracking, savings goals, and dashboard analytics.

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
- bcrypt

## Features
- User Authentication (Register/Login)
- Income Management
- Expense Management
- Budget Tracking
- Savings Goals
- Dashboard Analytics
- Secure JWT-based Authorization

## API Endpoints
- /api/auth
- /api/transactions
- /api/budgets
- /api/goals
- /api/dashboard
